# Tela de Login — README

Este repositório contém um componente de Tela de Login pronto para uso em aplicações web e móveis. O objetivo é fornecer uma interface elegante, acessível e fácil de integrar, com validação básica, tratamento de erros e estilo responsivo.

# Funcionalidades
Campos para e-mail/usuário e senha.
Validação do formulário (ex.: formato de e-mail, campo obrigatório, comprimento mínimo da senha).
Exibição de mensagens de erro e status (login em progresso, sucesso, falha).
Suporte a autenticação com token (ex.: JWT) ou callback para integração com qualquer backend.
Opção “Lembrar-me” e link para recuperação de senha.
Responsivo e adaptável para mobile e desktop.
Acessibilidade básica (labels, foco visível, atributos ARIA).
Componentização e fácil customização de estilos.
Tecnologias
HTML5 / CSS3 (ou framework CSS como Tailwind / Bootstrap)
JavaScript / TypeScript
